# Ayush Prajapati Portfolio

A modern and responsive personal portfolio website built with React.js and Tailwind CSS to showcase my skills, projects, education, and contact information.

## Features

* Responsive design for mobile, tablet, and desktop
* Modern UI with smooth navigation
* About Me section
* Skills showcase
* Projects gallery with live demo and GitHub links
* Testimonials section
* Contact form powered by Formspree
* Resume download functionality

## Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* React Icons
* Formspree
* Vercel
